# Electro Runner 3D

Небольшая 3D браузерная игра на `three.js`, полностью статическая и подходит для размещения на GitHub Pages.

## Что внутри

- управление персонажем в 3D-сцене (движение + прыжок);
- стартовый экран и кнопка повторной игры;
- мобильные touch-кнопки (влево/прыжок/вправо);
- сбор энерго-кубов для очков;
- препятствия и экран проигрыша;
- встроенная ссылка на бесплатный вебинар по проектированию электрики;
- favicon и мета-теги для красивого предпросмотра.

## Локальный запуск

Вариант 1: просто открой `index.html` в браузере.

Вариант 2 (рекомендуется):

```powershell
python -m http.server 8080
```

После этого открой `http://localhost:8080`.

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub и загрузи туда файлы проекта.
2. Открой `Settings -> Pages`.
3. В разделе `Build and deployment` выбери:
   - `Source: Deploy from a branch`
   - `Branch: main` и папка `/ (root)`
4. Сохрани настройки и подожди 1-2 минуты.
5. Сайт появится по адресу вида `https://<username>.github.io/<repository>/`.
